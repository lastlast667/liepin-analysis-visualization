## 一、功能概述

**岗位搜索模块**帮助您快速从海量招聘信息中筛选出符合条件的岗位。您可以通过关键词、城市、薪资、学历、经验等多维度条件进行精准搜索，查看岗位详情，并收藏感兴趣的职位。

**主要功能**：
- 🔍 多维度组合筛选（关键词 + 城市 + 薪资 + 学历 + 经验）
- 📋 卡片式岗位列表，信息一目了然
- ⭐ 一键收藏心仪岗位
- 📊 右侧热门统计面板（热门岗位、热门城市、热门公司）
- 📄 岗位详情页展示完整信息
- 💰 薪资分析对比
- 🔗 相似岗位智能推荐

## 二、岗位搜索列表页

### 2.1 如何使用搜索栏

页面顶部有一排搜索条件，从左到右依次为：

#### 第一步：输入关键词
在第一个输入框中，您可以输入：
- 岗位名称（如：Python开发、数据分析师）
- 公司名称（如：腾讯、阿里巴巴）
- 技能关键词（如：Spark、机器学习）

> 💡 **提示**：关键词支持模糊匹配，不需要输入完整名称。

#### 第二步：选择城市
点击"城市"下拉框，选择您期望的工作城市。下拉框中的城市列表来自平台现有的全部岗位数据。

> 💡 **提示**：默认值为"全国"，即不限制城市。

#### 第三步：选择薪资范围
点击"薪资范围"下拉框，选择您期望的薪资区间：
- 不限（默认）
- 10k 以下
- 10-20k
- 20-30k
- 30-50k
- 50k 以上

#### 第四步：选择学历要求
点击"学历要求"下拉框，选择您的学历水平：
- 不限（默认）
- 专科
- 本科
- 统招本科
- 硕士
- 博士

#### 第五步：选择经验要求
点击"经验要求"下拉框，选择您的工作经验：
- 不限（默认）
- 实习生
- 应届生
- 1-3年
- 3-5年
- 5-10年

#### 第六步：点击搜索
设置好所有条件后，点击最右侧的 **搜索按钮** ，系统将根据您的条件筛选并展示结果。

> ⚠️ **注意**：页面没有"重置"按钮，如需修改条件，直接在下拉框中重新选择即可。

---

### 2.2 如何查看搜索结果

搜索完成后，页面中间区域会展示岗位卡片列表。

**每个岗位卡片包含以下信息**：
| 信息项 | 说明 |
|--------|------|
| 岗位标题 | 蓝色可点击文字，点击后进入岗位详情页 |
| 公司名称 | 发布该岗位的公司 |
| 薪资 | 岗位提供的薪资范围 |
| 所在城市 | 岗位工作地点 |
| 学历要求 | 该岗位对学历的要求 |
| 经验要求 | 该岗位对工作经验的要求 |
| 招聘人数 | 该岗位计划招聘的人数 |
| 公司行业 | 公司所属行业领域 |

**卡片右上角**：有一个星形图标 ⭐，用于收藏/取消收藏该岗位。

**鼠标悬停效果**：将鼠标移动到卡片上时，卡片会有轻微的上浮和阴影效果，提示该卡片可交互。

---

### 2.3 如何排序与翻页

#### 排序方式
在搜索结果区域的右上角，有一个"排序"下拉框，支持以下排序方式：

| 排序选项 | 说明 |
|----------|------|
| 综合排序（默认） | 根据岗位标题与关键词的匹配度排序，最相关的排在最前面 |
| 薪资水平 | 按薪资从高到低排序 |
| 经验要求 | 按经验要求排序 |
| 学历要求 | 按学历要求排序 |

> 💡 **提示**：切换排序方式后，列表会立即重新加载，筛选条件保持不变。

#### 翻页操作
- 每页最多展示 **10 个岗位卡片**
- 页面底部有翻页控件，支持：
  - 点击具体页码跳转
  - 点击"上一页"/"下一页"切换
  - 快速跳转到首页或尾页

> 💡 **提示**：翻页时，您设置的筛选条件和排序方式都会被保留。

---

### 2.4 如何使用右侧热门面板

页面右侧有一列纵向排列的统计卡片，**不受顶部筛选条件影响**，始终展示平台全部数据的统计结果。

#### 卡片 1：热门岗位（薪资 Top 5）
展示平台中薪资水平最高的前 5 个岗位。
- 每条记录显示：序号 + 岗位标题 + 公司名称 + 薪资
- **点击岗位标题**：可直接进入该岗位的详情页

#### 卡片 2：热门城市（岗位数 Top 10）
展示岗位数量最多的前 10 个城市，格式为：`城市名（岗位数量）`
- 例如：`深圳（30）`、`北京（25）`、`上海（20）`...
- **点击城市标签**：会自动将该城市填入顶部搜索栏的"城市"下拉框，并立即触发搜索

> 💡 **提示**：这是快速筛选特定城市岗位的最快捷方式。

#### 卡片 3：热门公司（岗位数 Top 5）
展示发布岗位数量最多的前 5 家公司。
- 每条记录显示：公司名称 + 岗位数量
- **点击公司名称**：可进入该公司详情页（查看公司所有在招岗位）

---

### 2.5 如何收藏岗位

在岗位卡片的右上角，有一个星形收藏按钮：
- ⭐ **空心星**：表示未收藏，点击后变为实心星，岗位加入您的收藏列表
- ⭐ **实心星**：表示已收藏，点击后变为空心星，取消收藏

> ⚠️ **注意**：收藏功能需要登录账号。未登录时点击收藏按钮，系统会提示您先登录。

## 三、岗位详情页

点击列表页中任意岗位的标题，即可进入该岗位的详情页。

### 3.1 如何查看岗位详细信息

详情页顶部是一个大型信息卡片，展示该岗位的完整基础信息：

| 信息项 | 说明 |
|--------|------|
| 岗位标题 | 该岗位的具体名称 |
| 公司名称 | 发布公司 |
| 薪资 | 薪资范围 |
| 所在城市 | 工作地点 |
| 学历要求 | 学历门槛 |
| 经验要求 | 经验要求 |
| 招聘人数 | 计划招聘人数 |
| 公司行业 | 公司所属行业 |
| 公司规模 | 公司人员规模 |
| 工作制度 | "周末双休" 或 "弹性工作" |
| 岗位链接 | 原始招聘链接 |

---

### 3.2 如何申请岗位

在顶部信息卡片的右侧，有两个操作按钮：

| 按钮 | 功能 | 操作 |
|------|------|------|
| ⭐ 收藏 | 收藏/取消收藏该岗位 | 点击切换状态 |
| 🚀 立即申请 | 跳转到原始招聘页面 | 点击后在新标签页打开岗位链接 |

> 💡 **提示**："立即申请"会跳转到该岗位在招聘网站（如猎聘网）的原始页面，您可以在那里投递简历。

---

### 3.3 如何查看公司信息

详情页第三排是"公司介绍"卡片，展示：
- 公司名称
- 所属行业
- 公司规模
- 该公司在平台上的岗位数量
- 该公司在平台上的总招聘人数
- 该公司所有岗位的平均薪资（如：8.0k）

**卡片底部**：有一行文字链接 **"查看该公司所有岗位"**，点击后可进入公司详情页，查看该公司发布的全部岗位列表。

---

### 3.4 相似岗位推荐与薪资分析

详情页右侧同样有一列纵向面板：

#### 卡片 1：相似岗位推荐（Top 3）
系统根据当前岗位内容和您的浏览历史，智能推荐 3 个相似岗位。
- 每条推荐显示：岗位标题 + 公司名称 + 所在城市 + 薪资
- 当前阶段为展示功能，后续将支持点击跳转

#### 卡片 2：薪资分析
帮助您评估该岗位的薪资竞争力：
- **当前岗位薪资**：该岗位提供的薪资
- **行业平均薪资**：同行业所有岗位的平均水平
- **高于/低于平均百分比**：如"高于行业平均 15%"
- **行业薪资分布图**：展示行业最低薪资到最高薪资的范围，并标注当前岗位在分布中的位置

> 💡 **提示**：薪资分析数据基于平台全部数据计算，可帮助您判断该岗位的薪资是否具有竞争力。

---

### 3.5 如何返回搜索列表

在详情页的左上角，有一个 **"返回搜索结果"** 按钮。

- 点击后返回岗位搜索列表页
- 返回后，您之前的筛选条件、排序方式和页码都会保留，无需重新设置

> 💡 **提示**：如果您是通过"热门岗位"直接进入详情页，返回后会回到列表页的首页默认状态。

## 四、常见问题

### Q1：为什么搜索结果为空？
可能是筛选条件过于严格。建议：
1. 将某些下拉框改回"不限"
2. 检查关键词是否输入正确
3. 尝试扩大薪资范围或城市范围

### Q2：收藏的岗位在哪里查看？
当前版本收藏功能需要后端接口支持。前端 UI 已预留收藏按钮，完整功能将在后续版本上线。

### Q3：右侧热门面板的数据为什么和搜索结果不一致？
右侧热门面板展示的是平台**全部数据**的统计结果，不受顶部筛选条件影响。这样您可以随时了解全局热门信息。

### Q4：薪资分析中的"行业平均"是如何计算的？
行业平均薪资基于平台该行业所有岗位的 `month_salary_avg` 字段计算得出，反映该行业的整体薪资水平。

### Q5：相似岗位推荐是如何生成的？
系统采用多源混合推荐算法，综合考虑当前岗位内容相似度和您的历史浏览行为。当前阶段展示 Mock 数据，后续将接入真实推荐引擎。

### Q6：为什么有些公司信息在多个岗位中重复出现？
目前平台使用单表存储，公司信息随岗位记录冗余存储。后续将优化为公司独立表结构。

## 五、界面元素说明

### 5.1 图标含义

| 图标 | 含义 |
|------|------|
| ⭐（空心） | 未收藏，点击收藏 |
| ⭐（实心） | 已收藏，点击取消收藏 |
| 🔍 | 搜索按钮 |
| 🚀 | 立即申请，跳转外部链接 |
| ← | 返回上一页 |

### 5.2 颜色说明

| 颜色 | 含义 |
|------|------|
| 蓝色文字 | 可点击链接（岗位标题、公司名称、城市标签等） |
| 灰色文字 | 辅助说明信息 |
| 红色/高亮 | 重要数据或警告提示 |

### 5.3 状态提示

| 状态 | 说明 |
|------|------|
| 加载中 | 数据请求时显示旋转动画 |
| 暂无数据 | 筛选条件无匹配结果时显示 |
| 请求失败 | 网络异常时显示，可点击重试 |
